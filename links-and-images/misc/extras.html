HTML Entities

THere are three reserved characters in HTML: < > and &. You should always use HTML entities for these three characters

If you’re into “web typography,” You’ll also find yourself using curly quotes quite a bit.

Unlike straight quotes, these curly quote entities should hug the text.

Nowadays, we can insert any character directly into the HTML doc. Entities are mainly useful for reserved characters or convenience when writing straight HTML.
